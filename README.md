# baekjoon
This is an auto push repository for Baekjoon Online Judge created with [BaekjoonHub](https://github.com/BaekjoonHub/BaekjoonHub).


# judge.py
사용방법
1. 문제의 테스트 케이스를 input.txt 와 output.txt에 입력한다. 각 테스트 케이스 구분은 '###'로 한다.
2. solution.cpp에 본인의 코드를 입력한다.
3. 터미널에서 `python3 judge.py` 를 입력해서 파이썬 코드를 실행한다.

- solution.cpp 파일을 g++로 자동 컴파일 해주고, 나온 solution.exe에 input.txt의 값들을 입력. 나온 출력값들을 output.txt와 비교.
- 각 테스트케이스의 구분은 '###' 로 구분.
- python colorama 모듈 필요 (필수는 아님, 원치 않으면 judge.py의 로그 색깔 부분 수정 후 사용가능)
